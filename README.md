# pocapp

## Installation & run

Install NodeJS and npm
http://nodejs.org/

Get the project's dependencies via npm (see package.json for details):
```bash
npm install -g bower http-server
npm install
```

Get the project's Bower dependencies (see bower.json for details):
```bash
bower install
```

Launch/run the server (http://localhost:8080/):
```bash
http-server
```
